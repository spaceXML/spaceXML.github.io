오라클 Database (1)

구글 - 오라클 - 회원가입 - 다운로드
다른로드 목록
시작- sql-plus

select username from all_users;

입력

바탕화면 폴더만들기; dba 접속 명령어 최고관리자 명령어

ip 주소 확인하기 질의 작성기로 확인하기 빨간부분 ‘홑’ 으로 글자부분을 쓴다. ‘홑’ 부분에서는 대,소문자를 구분해서 쓴다.. Q1, EMP 테이블에서 전체 사원의 EMpno, ename, sal, deptno 값을 출력 where 부분에서 hiredate 에 81/04/02 까지 쪼개져서 나타나기 emp 테이블에서 empno, ename, sal, deptno 항목 모든 내용 조회하기 3번 EMP 테이블에서 empno가 7566번 사원의; 모든 컬럼을 출력 질문의 답이 맞았다. -

4번의 emp 테이블에서 empno가 7902번 사원의 sal 값만 출력 5번 DEPT 테이블에서 dname이 SALES인 모든 컬럼을 출력 하시오 ‘홑’ 속에 문자열은 대문자를 유지해서 적어야된다.

6번 DEPT 테이블에서 loc가 ‘DALLAS’인 deptno, dname 값 출력 –SALGRADE 테이블에서 GRADE 값이 3이상인 모든 컬럼 출력 하시오 데이터베이스에서는 문자형이 제일 많이나옴 –데이터 타입 조회 방법 위아래 조회는 같다.

트랜잭션이란

하나가 생기면 하나가 변하는 동시성을 뜻한다. SELect * –조회하기 위해 select 무조건 적는다. from EMP; – EMP 테이블로 부터 조회한다.

select * from dept;

select * from salgrade;

select ename, job, MGR – Ename, job, MGR컬럼의 내용을 조회 from emp;

select hiredate, sal, comm from emp;

select * from emp; where empno = 7782;

select * from emp where JOB = ‘salesman’;

select * from EMP WHere DEPTNO = 30;

select EMPNO, ENAME, JOB from EMP where JOB = ‘MANAGER’; –1) EMP 테이블에서 전체 사원의 empno, ename, sal, – deptno 값을 출력

select empno, ename, sal, deptno from EMP – 전체 조건은 where 에 안쓰면 된다.

select * from EMP where hiredate = ‘81/04/02’;

SCOTT계정에 쓴 내용들 SQL 내용

– 3번 3번 EMP 테이블에서 empno가 7566번 사원의; 모든 컬럼을 출력 질문의 답이 맞았다.

select * from EMP where hiredate = ‘81/04/02’;

–4번의 emp 테이블에서 empno가 7902번 사원의 sal 값만 출력 select sal from EMP where empno = 7902;

–5번 DEPT 테이블에서 dname이 SALES인 모든 컬럼을 출력 하시오 select * from DEPT where dname = ‘SALES’;
–6번 DEPT 테이블에서 loc가 ‘DALLAS’인 deptno, dname 값 출력 select deptno, dname from DEPT where loc = ‘DALLAS’;
–번 외 문제 –SALGRADE 테이블에서 GRADE 값이 3이상인 모든 컬럼 출력 select * from salgrade where Grade >= 3; – 연상자 : =, <, >, >=, <=

–데이터 타입 조회 DESC EMP; 오라클사용 Bat 파일로 클릭만으로 자동 실행되는 방법, 즉 자동실행 파일을 만든다.

net start OracleServiceXE net start OracleOraDB18Home1TNSListener TXT -> BAT으로 확장자 변환 종료가 된다면 자동 실행이 된다.
